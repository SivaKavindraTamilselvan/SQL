## Objective

Leverage window functions to perform calculations across a set of rows.

## 📋 Requirements

- [x]  Write a query using window functions such as `ROW_NUMBER()`, `RANK()`, or `DENSE_RANK()` to assign ranks (e.g., rank employees by salary within each department)
- [x]  Use `PARTITION BY` to define groups and `ORDER BY` to specify the ranking order
- [x]  Experiment with other window functions like `LEAD()` or `LAG()` to access adjacent row values
